# 💰 Expense Tracker

A simple **Expense Tracker application** built with Python.  
This project helps users record daily expenses, view spending history, and track total expenses easily.

## 📌 Features

- ➕ Add new expenses
- 📋 View all expenses
- ✏️ Edit expense details
- ❌ Delete expenses
- 💾 Save expense records
- 📊 Calculate total spending
- 🖥️ Easy-to-use interface

## 🛠️ Technologies Used

- Python 3
- File handling / JSON
- Command Line Interface (CLI)
